# Livewire Demo App (TALL Stack)

Демонстрационное приложение, реализованное в рамках тестового задания.

## 🚀 Стек технологий

- Laravel (latest)
- Livewire 4
- Alpine.js
- TailwindCSS
- Blaze-подход (оптимизация ререндера через wire:key)

---

## 📌 Реализованный функционал

### 1. Список элементов
- Отображение списка постов
- Данные сгенерированы (mock)

### 2. Lazy Loading (Infinite Scroll)
- Реализована подгрузка элементов при прокрутке страницы
- Использован IntersectionObserver (через Alpine.js)
- Без багов повторной загрузки и зацикливания

### 3. Computed Properties
- Реализована фильтрация (поиск)
- Реактивное обновление списка
- Динамический пересчет результатов

### 4. Event Dispatch
- При клике на карточку генерируется событие
- Готово для взаимодействия между компонентами

### 5. UI / UX
- Чистый адаптивный интерфейс
- Отсутствие "мигания" при загрузке
- Loader отображается только при подгрузке данных
- Корректная обработка состояний:
    - Loading
    - No results
    - No more posts

---

## ⚙️ Особенности реализации

- Оптимизирован ререндер через `wire:key`
- Исключены лишние запросы
- Реализован стабильный infinite scroll (без багов IntersectionObserver)
- Разделена логика:
    - полный список (`posts`)
    - отфильтрованный список (`filteredPosts`)

---

## 📷 Основные сценарии

- Загрузка данных при скролле
- Поиск по списку
- Динамическое обновление UI без перезагрузки
